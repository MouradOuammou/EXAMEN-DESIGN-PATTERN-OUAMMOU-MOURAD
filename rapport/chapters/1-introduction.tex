\chapter{Introduction}

\section{Contexte}
Ce projet vise à démontrer l'application pratique des design patterns en Java.

\section{Objectifs}
\begin{itemize}
    \item Implémenter les principaux design patterns
    \item Créer une application JavaFX pour visualiser ces patterns
    \item Documenter l'architecture et les décisions de conception
\end{itemize}
