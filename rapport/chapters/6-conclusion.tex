\chapter{Conclusion}

\section{Résumé des réalisations}

Ce projet a démontré l'implémentation pratique de cinq design patterns majeurs en Java.

\section{Apprentissages}

\begin{itemize}
    \item Compréhension approfondie des patterns
    \item Utilisation de Maven et AspectJ
    \item Développement d'interfaces JavaFX
    \item Tests unitaires et couverture de code
\end{itemize}

\section{Perspectives futures}

\begin{itemize}
    \item Ajouter plus de patterns (Factory, Singleton, Decorator)
    \item Développer une interface web avec Spring Boot
    \item Améliorer la couverture des tests
\end{itemize}
